SkillsBarChart

Hi there! This is a small component written with d3 that can be used to show off your skills. You know, like nunchuck skills, bow hunting skills, computer hacking skills. Girls only want boyfriends who have great skills!

Demos

This library is exposed as a UMD (universal module definition), i.e. can be used as a global variable added to the window object or with most module loaders.

Here is the usage of this library in various ways:

  • As a global variable SkillsBarChart
  • As a commonJS module using browserify
  • As an ES6 module in a webpack environment
  • As a typescript module in an angular project
